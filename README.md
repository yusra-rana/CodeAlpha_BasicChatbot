# CodeAlpha Basic Chatbot

## Description

A simple rule-based chatbot developed in Python for the CodeAlpha Python Programming Internship.

## Features

* Responds to greetings
* Answers simple questions
* User-friendly console interaction
* Exit option using "bye"

## Technologies Used

* Python
* Functions
* Loops
* Conditional Statements

## How to Run

```bash id="6e72ei"
python chatbot.py
```

## Example

```text id="6q6q0e"
You: hello
Bot: Hi! Nice to meet you.

You: bye
Bot: Goodbye! Have a great day.
```
